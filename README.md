# 🌍 Travel Assistant Bot

Telegram-бот для получения информации о погоде и прогнозе погоды в разных городах.

## 📌 Возможности

- 🌤 Получение текущей погоды
- 📅 Прогноз погоды на 3 дня
- 📜 История запросов
- ⭐ Сохранение любимого города
- 🗑 Очистка истории
- 📊 Статистика запросов
- 🤖 ИИ-совет путешественнику
- ℹ️ Справка по командам

## 🛠 Используемые технологии

- Python
- pyTelegramBotAPI
- Open-Meteo API
- DeepSeek API
- JSON
